# Python Taschenrechner

Konsolen-Taschenrechner in Python als Lern- und Übungsprojekt.

## Projektidee
Das Projekt dient dazu, grundlegende Python-Konzepte praktisch zu üben und schrittweise auszubauen.  


## Aktueller Stand
- Eingabe von Zahlen über die Konsole
- Grundrechenarten: Addition, Subtraktion, Multiplikation, Division
- Wiederholte Ausführung über eine Schleife (`while`)
- Beenden des Programms mit Eingabe von `q`
- Fehlerbehandlung bei ungültigen Zahleneingaben

## Dateien
- `taschenrechner.py`  
  Erste einfache Version
- `taschenrechner_mit_operatoren.py`  
  Erweiterung um mehrere Rechenoperatoren
- `taschenrechner_loop.py`  
  Aktueller Entwicklungsstand mit Schleife und Abbruchbedingung
- `taschenrechner_kommentiert.py`  
  Kommentierte Lernversion zur besseren Nachvollziehbarkeit
